# 🚀 Interactive Pushdown Automata (PDA) Simulator

An advanced, web-based simulation tool to visualize the operations of a Pushdown Automaton. Built to simplify Theory of Computation concepts, this project demonstrates how PDAs recognize context-free languages (like $a^n b^n$) through real-time state transitions and stack memory operations.

## ✨ Key Features
* **Dynamic State Machine Map:** Automatically generates a live node-graph of your automaton using Graphviz. The active state highlights dynamically as you step through the input!
* **Visual Stack Memory:** Watch the LIFO (Last-In-First-Out) stack update in real-time as `push` and `pop` operations execute.
* **Step-by-Step Execution:** Control the flow of computation character by character to easily trace the machine's logic.
* **Custom Transitions:** Fully customizable state definitions, input symbols, and stack alphabets to test any Context-Free Grammar.
* **Modern UI:** Clean, glassmorphism-inspired interface built entirely in Python.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Web Framework:** Streamlit
* **Visualization:** Graphviz
